## 创建项目

### 使用`create-react-app`创建项目

#### 安装create-react-app 

> 如果之前安装过，最好先卸载掉，重新安装

<!-- 卸载 -->
```
- npm uninstall -g create-react-app
- yarn global remove create-react-app

```

#### 使用npm创建项目
```
npx create-react-app my-app-ts --template typescript
```

#### 启动

```
yarn start
```
本地访问：http://localhost:3001/
